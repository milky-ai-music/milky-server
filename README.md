# Milky🌌
Milky는 사용자 정보(위치, 선호 음악 장르, 무드)를 바탕으로 AI에게 음악을 추천받아 사용자에게 음악을 들을 수 있는 환경을 제공하는 서비스입니다.

## Requirements
- Nest.js (Node v22)
- MariaDB

## Install Guide
1. git clone https://github.com/milky-ai-music/milky.git
2. cd ../package.json
3. npm install
4. npm run start
